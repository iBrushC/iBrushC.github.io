About


My name is Andrew Combs, and I'm a computer scientist, artist, game developer, chef/baker, free-runner... you get the point. I love challenging myself to learn new things and see how I can adapt them. This site focuses more on my coding and art related work, but I plan to make another website soon, just for the less technical things I do.



When I was eight, I was introduced to Scratch, an online website for making games by connecting blocks. I got hooked, and to this day I've kept coding. Python was my first text-based language, and since then I've picked up a few more, including C, C++, Javascript, Rust, and a handful of shading languages.



While I enjoy all aspects of programming, I'm most fond of anything that has some visual output. Any kind of simulation, generative art, or image processing isn't just an excellent challenge, but is an extremely rewarding one too! Unlike data science or most algorithm work, good visual programming gives a remarkable output that anyone can see the beauty in. It's much easier to show someone how incredible programming is with a good fractal than a good search algorithm.
